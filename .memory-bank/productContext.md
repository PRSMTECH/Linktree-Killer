# Product Context - Social Links Landing Page

## Project Purpose
A "link in bio" style landing page that consolidates all of @MrJPTech's social media platforms in one place. Designed for use across Instagram, TikTok, Twitter, and other platforms where bio links are limited to a single URL.

## Target Audience
- Social media followers wanting to find all platforms
- New viewers discovering @MrJPTech content
- Business contacts looking to connect

## Core Features
1. **Profile Section** - Avatar with PRSMTECH logo, name, title
2. **8 Social Platform Links**:
   - Twitch: @MrJPTech
   - YouTube: @MrJPTechy
   - TikTok: @MrJPTech
   - Instagram: @MrJPTech (Personal)
   - Instagram: @prsmtech (Company)
   - X/Twitter: @MrJPTech
   - Threads: @mrjptech_
   - Reddit: u/MrJPTech
3. **Website CTA** - Link to prsmtechweb.com
4. **Animated UI** - Framer Motion animations, glass morphism
